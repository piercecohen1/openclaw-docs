# OpenClaw Docs Mirror

Local mirror of all OpenClaw documentation from docs.openclaw.ai, stored as raw markdown under `docs/`. See `README.md` for scrape date, page counts, and directory tree.

## How It Works

- Single `llms.txt` index at `https://docs.openclaw.ai/llms.txt` lists all pages as markdown links
- `download.sh` fetches the index, extracts `.md` URLs, downloads all pages preserving directory structure
- Also downloads `llms-full.txt` — single-file concatenation for full-text search
- The README is auto-generated by `download.sh` with scrape metadata, section counts, and directory tree

## Re-scraping

```bash
bash download.sh
git add -A && git commit -m "Update docs mirror $(date +%-m-%-d-%y)" && git push
```

## Searching

```bash
rg "query" docs/                      # All docs
rg "query" docs/llms-full.txt         # Full-text search
```
